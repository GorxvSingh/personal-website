# Phase 1: Foundation — Plan 01

## Objective

Set up Expo project with TypeScript, React Navigation, and development tooling. Establish the project structure that all future phases will build on.

## Execution Context

**Mode:** YOLO (auto-approve)
**Depth:** Standard

**Prior Decisions:**
- Expo managed workflow (faster iteration)
- iOS only for v1
- TypeScript with strict mode

## Tasks

### Task 1: Initialize Expo Project

Create new Expo project with TypeScript template.

```bash
npx create-expo-app@latest ai-or-not --template expo-template-blank-typescript
cd ai-or-not
```

**Verify:** `npx expo start` launches without errors

---

### Task 2: Configure TypeScript Strict Mode

Update `tsconfig.json` for strict TypeScript:

```json
{
  "extends": "expo/tsconfig.base",
  "compilerOptions": {
    "strict": true,
    "noImplicitAny": true,
    "strictNullChecks": true,
    "noUnusedLocals": true,
    "noUnusedParameters": true
  }
}
```

**Verify:** No TypeScript errors in existing files

---

### Task 3: Install React Navigation

Install navigation dependencies:

```bash
npx expo install @react-navigation/native @react-navigation/stack react-native-screens react-native-safe-area-context react-native-gesture-handler
```

**Verify:** Dependencies added to package.json

---

### Task 5: Define Core Types

Create TypeScript types from PROJECT.md data model:

**File:** `src/types/index.ts`

```typescript
export interface ImageCard {
  id: string;
  imageUrl: string;
  isAi: boolean;
  difficulty: 'easy' | 'medium' | 'hard';
  metadata: {
    photographerName?: string;
    sourceUrl?: string;
    aiModel?: string;
    tells?: string[];
  };
}

export type SwipeDirection = 'left' | 'right';
export type GuessResult = 'correct' | 'incorrect';
```

**Verify:** Types compile without errors

---

### Task 6: Configure ESLint and Prettier

Add linting and formatting:

```bash
npm install --save-dev eslint @typescript-eslint/parser @typescript-eslint/eslint-plugin prettier eslint-config-prettier
```

**Create:**
- `.eslintrc.js` — ESLint config with TypeScript rules
- `.prettierrc` — Prettier config (single quotes, no semicolons, trailing commas)

**Verify:** `npm run lint` passes (add script to package.json)
